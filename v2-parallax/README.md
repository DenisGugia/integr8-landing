# Integr8 Landing Page v2 — Parallax

## Como usar

Abrir `index.html` no browser. Funciona como arquivo local ou servido.

## Substituir imagens

As imagens de fundo estão em `images/`. Para trocar:

1. Coloque a nova imagem na pasta `images/`
2. No `index.html`, busque o nome do arquivo antigo e substitua
3. Imagens ideais: landscape (16:9), resolução mínima 1920×1080
4. As imagens terão overlay escuro (40-72%), então precisam de bom contraste

## Substituir links

Buscar e substituir no `index.html`:
- `#checkout` → Link do Stripe
- `#whatsapp` → `https://wa.me/SEU_NUMERO`
